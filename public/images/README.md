# Portfolio Images

This directory contains all images used in your portfolio.

## Profile Image

Add your profile image here as `profile.png`:
- Place your PNG image file in this directory
- Name it `profile.png` 
- Recommended size: 400x400px or larger (square format works best)
- Used in the About section

## Project Images

Add project preview images here:
- `project1.png` - NinjaLingo project
- `project2.png` - CMU Share Cycle project
- Recommended size: 400x300px or larger (landscape format)
- Can use PNG, JPG, JPEG, or WebP formats

## Directory Structure

```
public/
└── images/
    ├── profile.png      (your profile photo)
    ├── project1.png     (NinjaLingo project image)
    └── project2.png     (CMU Share Cycle project image)
```

All images are automatically styled with borders and shadows to match the mystical theme!
